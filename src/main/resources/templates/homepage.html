About Us

Your Trusted Partner in Reading Journey

BeSmart is a comprehensive bookstore management system built with Java Spring Boot, implementing full CRUD (Create, Read, Update, Delete) operations with robust object-oriented programming principles.

  • Abstract Class implementation for Person and MediaCetak entities.
  • Interface implementation for Diskon.
  • Associated classes (Penjual, Pembeli, MediaCetak entities).
  • Database CRUD operations using JPA/Hibernate ORM
